# Todo App (Express + EJS)

## Overview
This project is a simple Todo application built with Node.js and Express.  
Users can create, update, delete, and manage tasks through a web interface. The application also calculates and displays the achievement rate based on completed tasks.

![image alt](https://github.com/Ferb-10/Todo-18/blob/5c0597a19b74e1770a9337ec6d44fe3514ef82e9/public/Todo-home.png)

## Features
- Create new tasks (Add)  
- Update task status (Toggle completion)  
- Edit existing tasks  
- Delete tasks  
- Achievement rate calculation based on completed tasks  
- Dynamic date display using custom middleware  

## Technologies Used
- Node.js  
- Express  
- EJS 
- HTML / CSS  

## What I Learned
- How to implement full CRUD functionality in a web application  
- Using Express routing for handling different operations (GET/POST)  
- Managing application state with arrays and objects  
- Passing data to views using EJS and res.render  
- Utilizing middleware to inject reusable data (e.g., date)  
- Calculating and displaying dynamic values (achievement rate)
